Biblioteca

¿De que trata?

La base de datos está en el archivo biblioteca.libros.js, para importarlo en el MongoDBcompass. El servidor se ejecuta usando node server.js en la terminal.

La web accede a una base de datos con una lista de libros. Los libros cuentan con una ID unica, un titulo, una imagen y su genero literario. Los libros pueden ser filtrados por autor o genero. Tambien se puede buscar un libro por su ID o por su titulo.

Ademas, la informacion de los libros puede ser modificada. Esto quiere decir que tanto su titulo, imagen y genero pueden actualizarse.

Por otro lado, tambien es posible eliminar cualquiera de los libros que se desee.

Por ultimo, la web cuenta con un filtro para ordenar todos los libros por orden alfabético. Muy util para volver a mostrar todos los libros en caso de haber filtrado anteriormermente por genero o autor.
